1. Introduction

Script Kittens ("we", "our", "us") operates the Script Kittens Discord Bot. This Privacy Policy explains what data we collect, why we collect it, how it is stored, and your rights regarding your data. By using the Bot you consent to the practices described here.

2. Data We Collect

We collect only the minimum data necessary to operate the Bot's features:

Data Why We Collect It How Long We Keep It
Discord User ID Rate limiting, premium checks, key system, command logging Until manually deleted or account removed
Discord Username Display purposes in logs and admin commands Until manually deleted
Discord Guild (Server) ID Per-server configuration (channel restrictions, ticket settings, leveling) Until bot is removed from server
Discord Channel ID Channel restriction settings per command type Until configuration is cleared
Command Usage Logs Debugging, abuse prevention, analytics Up to 90 days
Rate Limit Records Enforcing per-user command usage limits Automatically deleted when the limit window expires
Premium Subscription Records Verifying active premium status and expiry dates Until subscription expires or is removed
Access Keys Authenticating users in the key-based access system Deleted on expiry (24 hours) or on use
Moderation Records (warns) Server moderation — storing warnings issued by server staff Until cleared by server admin or bot removal
XP / Level Data Leveling system — tracking activity per user per server Until cleared or bot removed

3. Data We Do NOT Collect

  • Message content (the Bot reads messages only to detect bot commands; it does not store message text).
  • Passwords, payment card numbers, or financial information.
  • Voice data.
  • Email addresses or any personally identifiable information beyond your Discord User ID and Username.

4. How We Store Data

All data is stored in a MongoDB database hosted on a private, password-protected cloud cluster. Access to the database is restricted to authorized operators only. Data is not shared with or sold to any third parties.

5. Third-Party Services

Certain Bot commands query external Free Fire game data APIs. When you use commands such as /info, /bancheck, /visit, or /stats, the player UID you provide is sent to these third-party APIs. We do not control those APIs and are not responsible for their privacy practices. No Discord user data is sent to these APIs.

6. Data Sharing

We do not sell, trade, rent, or share your personal data with any third party except:

  • When required by law or a valid legal process.
  • To protect the rights, property, or safety of Script Kittens, its users, or the public.

7. Your Rights

You have the right to:

  • Access — request a summary of what data we hold about you.
  • Deletion — request that your data be deleted. We will process deletion requests within 30 days. Note: deleting your data may disable certain Bot features for your account.
  • Correction — request that incorrect data be corrected.

To exercise any of these rights, open a support ticket in the Script Kittens Discord server.

8. Children's Privacy

The Bot is not directed at children under 13. We do not knowingly collect data from users under 13. If you believe a child under 13 has used the Bot, contact us and we will delete the relevant data.

9. Changes to This Policy

We may update this Privacy Policy at any time. We will notify users of significant changes via our Discord server. Continued use of the Bot after changes are posted constitutes acceptance of the updated Policy.

10. Contact

For any privacy-related questions or data requests, open a support ticket in the Script Kittens Discord server.
